# Road Game

### Install libraries 

```bash
pip install -r requirements.txt
```

### deploy virtual environment

```bash
python -m venv venv
```

### Activation for **windows**:

```bash
venv/scripts/activate
```

### run pygame

```bash
python game/main.py
```

### built html

```bash
pygbag --build game
```
